18학번 정진선, 김유림 스크립트언어 최종발표 시작하겠습니다
발표는 활용 데이터, 구현된 기능, 주차별계획, 깃허브, 배포파일생성, 시연 순으로 진행하겠습니다

활용데이터는 학과정보 Open API와 지도 연동을 위해 Google Map API를 사용하였습니다

구현된 기능은 
저번 발표때까지 전체 UI를 구성하고 파싱한 데이터로 검색, 카테고리화, 리스트화, 데이터 도식화, 지도연동을 구현하였습니다. 또한 C++을 연동해 지역에 따라 대학을 분류하였습니다. 그 이후로 Gmail전송과 챗봇, 추가로 브라우저를 구현하였습니다

Gmail과 연동하여 사용자로부터 메일 주소를 입력받아 프로그램내에 저장된 정보를 사용자에게 전송하도록 구현했습니다.
Telegram으로 봇을 생성하여 토큰과 UserID를 가져와 다음과 같이 취업, 어디까지 알아봤니 봇을 구현하였습니다. 
추가 구현으로 사람인 브라우저를 Tkinter에 적용하였습니다.

다음은 주차별 계획입니다. 
다음과 같이 4주차로 계획했던 것에 브라우저 기능을 추가로 구현하면서 계획이 일부 수정되었고 19일에 마지막 챗 봇을 구현하면서 계획을 전부 수행하였습니다. 

깃허브는 지난 4주간 다음과 같은 횟수로 커밋이 되었고 이번 프로젝트 동안 총 55회의 커밋이 이루어졌습니다.

다음은 배포파일을 생성하겠습니다.
(shift + 오른쪽 마우스 클릭) -> powershell -> python.exe setup.py sdist -> 폴더내에 dist  파일 생성 확인후 압축해제 -> powershell - > python.exe setup.py install -> Python38 폴더내의 Lib 폴더에  site-package 에 생성된것 확인
